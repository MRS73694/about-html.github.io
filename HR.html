The HR Tag and its Attributes

Attribute Use
Width Width Attribute displays the horizontal ruler of the specified width in pixels or in percentage
Size Size Attribute controls the height or the thickness of the horizontal ruler
Note: The size attribute takes the value in pixels
Color Color Attribute accepts the name of the color or hexadecimal code to apply as the background colour of horizontal ruler
Noshade Noshade Attribute displays the ruler bar as a flat solid bar rather than the default 3D bar
Note: This attribute is used without any value
Align Align Attribute can take any one of the 3 alignment values, that is left, center or right to align the position of horizontal ruler
Note: The effect of the align attribute is visible only if the web width of the hr tag is less than the width of the web browser window
